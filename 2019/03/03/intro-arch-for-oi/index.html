在安装之前,请确保你的PC\text{PC}PC中有足够的硬盘空间存放Arch Linux\text{Arch\ Linux}Arch Linux的根目录,并划分出根分区,建议划出大于等于你的PC\text{PC}PC的运行内存的swap\text{swap}swap分区,分配出50MB\text{50MB}50MB到200MB\text{200MB}200MB不等的ESP\text{ESP}ESP分区(如果Windows\text{Windows}Windows已经创建了可以使用]Windows]\text{Windows}]Windows创建的.以上操作建议在安装之前完成(完成之后可以忽略文中分区部分).

Start Installing

Arch Linux\text{Arch\ Linux}Arch Linux以滚动更新的形式发布,它能给用户提供较为基础和可自定义的Linux环境,并且始终让用户通过滚动更新获得最新的软件包.它使用名为pacmanpacmanpacman(Package Manager吃豆人)软件包管理器来管理软件包,只需一个命令，整个操作系统中所有的软件包都会开始更新到最新的版本(然而这也有一个问题,那就是更新某些系统组件会带来一些内核级别的错误,让你的Arch Linux\text{Arch\ Linux}Arch Linux瞬间Kernel Panic\text{Kernel\ Panic}Kernel Panic),正是因为这一点,我们可以使用Arch Linux\text{Arch\ Linux}Arch Linux获得最有效率逼格的Linux\text{Linux}Linux体验.因为滚动更新的特性,Arch Linux\text{Arch\ Linux}Arch Linux的安装ISO\text{ISO}ISO也采用了快速更新发布,每月更新一次.对于中国大陆的用户,你可以在tuna\text{tuna}tuna源下载到Arch Linux\text{Arch\ Linux}Arch Linux的安装ISO\text{ISO}ISO(当然，在其他途径下载仍然可以)
